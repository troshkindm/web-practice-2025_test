# Изображения для блюд

Текущие изображения - это SVG placeholder'ы.

Для замены placeholder изображений на настоящие:

1. Сгенерируйте изображения блюд через нейросеть (например, Midjourney, DALL-E, Stable Diffusion)
2. Сохраните их с теми же именами:
   - Супы: soup1.jpg, soup2.jpg, soup3.jpg
   - Главные блюда: main1.jpg, main2.jpg, main3.jpg, main4.jpg, main5.jpg, main6.jpg
   - Напитки: drink1.jpg, drink2.jpg, drink3.jpg
3. Замените файлы в этой папке

Рекомендуемый размер изображений: 800x600px или больше.
